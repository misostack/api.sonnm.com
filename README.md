# CMS feature

> Version: 1.0

Includes:

1. User Module

- Manage users
- Static user roles : super admin, admin
- Only 1 super admin
- Public API 2 factor login

2. Post Module

- Manage Custom Content Types
- Manage Categories
- Public API Categories
- Manage Post Type
- Public API List Posts by Content Type
- Public API Post Detail

3. File Module

- Attachment Upload
- Attachment View/Download
